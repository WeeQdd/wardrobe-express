# WardrobeExpress Bot

Telegram-бот для автоматизации заявок на выкуп и доставку товаров.

Бот работает как посредник между клиентом и администратором:
- принимает заявки на товары по ссылке и из каталога `В наличии`
- объединяет несколько товаров в одну заявку
- сохраняет телефон, адрес доставки и выбранный ПВЗ пользователя
- подбирает ближайшие ПВЗ СДЭК по адресу или геопозиции
- рассчитывает стоимость товара с учетом валюты, наценки и доставки по России
- отправляет оформленные заявки администраторам
- позволяет администратору менять статусы заказов и писать клиенту через бота

## Переменные окружения

Минимально нужны:

```env
TOKEN=your_telegram_bot_token
DB_URL=sqlite+aiosqlite:///data/my_base.db
YANDEX_GEOCODER_API_KEY=your_yandex_api_key
```

Дополнительно при необходимости:

```env
ADMIN_GROUP_ID=your_admin_group_id
API_KEY=your_currency_api_key
```

## Запуск через Docker

### 1. Собрать образ

```bash
docker build -t wardrobeexpress-bot .
```

### 2. Подготовить `.env`

Создайте файл `.env` в корне проекта и заполните нужные переменные.

Для SQLite в контейнере лучше использовать путь в отдельной папке:

```env
DB_URL=sqlite+aiosqlite:///data/my_base.db
```

### 3. Запустить контейнер

```bash
docker run -d \
  --name wardrobeexpress-bot \
  --env-file .env \
  -v ./data:/app/data \
  wardrobeexpress-bot
```


- При первом запуске бот сам создаст таблицы и заполнит базовые данные.
- Локальная база SQLite хранится в примонтированной папке `data`.
- Если не примонтировать том, база будет жить только внутри контейнера.
